# Задание 10 (Kotlin)

Проект в папке `Task10` состоит из 3 модулей:

1. `library-console` — пункт 1 (консоль): библиотека книг с делегированием (читатели/админы).
2. `warehouse-console` — пункт 2 (консоль): склад и ленивые загрузки категорий (`by lazy`) + делегирование форматтера.
3. `room-android` — пункт 3 (Android): Room + KSP. На экране можно `Добавить`, `Загрузить` и `Очистить` таблицу.

## Что показать на скриншотах

1. Для `library-console`:
   - видно, что `Admin` добавляет книги,
   - видно, что `Reader` может `взять` и `вернуть` книгу (вторая попытка взять занятую книгу возвращает `false`).

2. Для `warehouse-console`:
   - при первом обращении к `company.fruits` выводится строка `LAZY: подгружаю товары категории FRUITS...`,
   - при втором обращении к `company.fruits` этой строки больше не появляется,
   - при первом обращении к `company.drinks` появляется аналогичная строка для `DRINKS`.

3. Для `room-android`:
   - нажать `Добавить запись в Room` и увидеть `Добавлено: id=...` в `txtOutput`,
   - нажать `Загрузить все записи` и увидеть список строк из таблицы.

## Запуск

- `library-console` и `warehouse-console`: запускай `Main.kt` из соответствующих модулей.
- `room-android`: запускай `MainActivity`.
